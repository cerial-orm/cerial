Decorators that create database indexes on individual fields.

| Decorator | Purpose |
|-----------|---------|
| [`@unique`](/schema/decorators/indexes/unique) | Unique constraint — reject duplicate values |
| [`@index`](/schema/decorators/indexes/index-decorator) | Non-unique index for query performance |
