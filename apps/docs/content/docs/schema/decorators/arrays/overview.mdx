description: Array-specific decorators for deduplication and sorting.
---

Decorators that control array field behavior at the database level.

| Decorator | Purpose |
|-----------|---------|
| [`@set`](/schema/decorators/arrays/set) | Auto-deduplicate and sort array values |
| [`@distinct`](/schema/decorators/arrays/distinct) | Deduplicate without sorting |
| [`@sort`](/schema/decorators/arrays/sort) | Sort array values |
