description: Field behavior control decorators for nullability, immutability, and flexibility.
---

Decorators that control how fields accept and store values.

| Decorator | Purpose |
|-----------|---------|
| [`@nullable`](/schema/decorators/control/nullable) | Allow explicit null values |
| [`@readonly`](/schema/decorators/control/readonly) | Write-once, excluded from updates |
| [`@flexible`](/schema/decorators/control/flexible) | Accept additional untyped properties on objects |
