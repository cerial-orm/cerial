description: UUID auto-generation decorators for server-side ID creation.
---

Decorators that auto-generate UUID values on record creation.

| Decorator | Purpose |
|-----------|---------|
| [`@uuid`](/schema/decorators/uuid/uuid) | Generate UUID v7 (default, time-ordered) |
| [`@uuid4`](/schema/decorators/uuid/uuid4) | Generate UUID v4 (random) |
| [`@uuid7`](/schema/decorators/uuid/uuid7) | Generate UUID v7 (explicit) |
