description: Relation decorators for linking models together.
---

Decorators that define relationships between models.

| Decorator | Purpose |
|-----------|---------|
| [`@field`](/schema/decorators/relations/field) | Specify the FK field for a forward relation |
| [`@model`](/schema/decorators/relations/model) | Specify the target model for a relation |
| [`@key`](/schema/decorators/relations/key) | Specify the target field for the FK lookup |
| [`@onDelete`](/schema/decorators/relations/on-delete) | Control cascade behavior on delete |
