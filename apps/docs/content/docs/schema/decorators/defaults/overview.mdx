Decorators that provide default values when fields are absent.

| Decorator | Purpose |
|-----------|---------|
| [`@default`](/schema/decorators/defaults/default) | Set default on creation when field is absent |
| [`@defaultAlways`](/schema/decorators/defaults/default-always) | Reset to value on every create and update |
