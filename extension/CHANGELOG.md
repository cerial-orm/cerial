# Changelog

All notable changes to the Cerial extension will be documented in this file.

## [0.0.1] - Initial Release

### Added

- Syntax highlighting with TextMate grammar and semantic tokens
- Smart completions for keywords, types, decorators, and cross-file references
- Real-time diagnostics with 11 schema validators
- Document formatting with 9 configurable style options
- Go-to-definition, find references, and rename across files
- Hover documentation with type info and SurrealDB mappings
- Code actions with 12 quick fix types
- Document outline and workspace symbols
- Folding ranges for all block types
- Inlay hints for FK types, server-set fields, and inherited fields
- Document links for cross-file type references
- 16 code snippets for common schema patterns
- Multi-schema workspace support via cerial.config.ts
